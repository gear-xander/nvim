# nvim

Simple Neovim config using `lazy.nvim`.

Requires `Neovim 0.11+`.

Dependencies:

- `git`
- `ripgrep` for Telescope `live_grep`
- `terraform` for `terraform fmt`

Includes:

- basic settings and keymaps
- file tree with `nvim-tree`
- fuzzy finding and live grep with `telescope.nvim`
- LSP for Lua, Python, Bash, and Terraform
- Treesitter syntax highlighting
- format on save
- Git signs
- `tokyonight` theme
